# Data, Model & Code License Manifest

_Required deliverable per the challenge brief ("a current data, model, and code license manifest"). Current as of the frozen event date 2026-07-18._

## Models

| Model | Where it runs | Role | License / terms | Training on our data |
|---|---|---|---|---|
| **None (deterministic engine)** | Local | ALL scored behavior: extraction, annualization, threshold comparison, readiness, citations | Our code, MIT-style hackathon submission | n/a |
| RapidOCR (PP-OCR ONNX models via `rapidocr-onnxruntime`) | **Local only** — no network | OCR for rasterized synthetic documents | Apache-2.0 | n/a — runs offline |
| OpenAI (`OPENAI_MODEL`, default `gpt-4o-mini`) | OpenAI API — **optional**, active only when `OPENAI_API_KEY` is set; disclosed on the consent screen | Non-authoritative assist only: question→intent routing, gated plain-language rephrasing, opt-in advisory cross-check. Can never produce a number, threshold, status, or decision | [OpenAI Business Terms](https://openai.com/policies/business-terms/) / API Platform terms | **No** — OpenAI does not train on API data by default; we send requests with default retention and no opt-in to training |

Data sent to OpenAI when enabled (and nothing else — see `docs/feature-manifest.md`): the renter's typed question; the already-computed answer + citation for rephrasing; and, only under the separate off-by-default `REALDOOR_LLM_CROSSCHECK` flag, allowlisted field values extracted from the **synthetic** documents. Uploaded PDFs are never sent.

## Data

| Dataset | Source | License / limit |
|---|---|---|
| FY 2026 MTSP income limits (frozen subset) | Organizer pack, from HUD MTSP tables | Public HUD data; frozen effective 2026-05-01; used only for the 60% threshold lookup |

| Synthetic household documents + gold fields | Organizer pack | Hackathon evaluation materials; entirely synthetic — no real renter data anywhere in the system |
| Frozen 11-rule corpus, checklists, Q&A, adversarial tests | Organizer pack | Hackathon evaluation materials |
| `demo/demo_pay_stub_lowquality.pdf` | Generated by `scripts/make_demo_doc.py` | Our synthetic demo asset (abstention demonstration); excluded from all evals |

No public dataset is used to profile applicants or infer protected traits (see `docs/feature-manifest.md`, "Features deliberately absent").
